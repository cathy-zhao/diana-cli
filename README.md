# diana-cli

## 安装依赖

```bash
npm install
```

## 启动

 ```bash
 npm run watch
 ```

## 编译

```bash
npm run compile
```

## 全局使用 diana 命令

执行`npm link`

## 可使用命令

- `diana init <registry-template> <project-name>`
- `diana config get`
- `diana config set registry <registry>`
- `diana config set type <users>`
